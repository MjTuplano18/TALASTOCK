## 🎉 What's Talastock?

Talastock is a modern inventory and sales management system built for Filipino SMEs. It features:

✅ Complete POS system with barcode scanner support  
✅ 5 payment methods (Cash, Card, GCash, PayMaya, Bank Transfer)  
✅ 4 discount types (%, Fixed, Senior/PWD, Custom)  
✅ Full refund/return processing  
✅ Comprehensive reports with Excel export  
✅ AI-powered insights and recommendations  
✅ Bulk import/export (CSV/Excel)  
✅ Mobile responsive design  
✅ Security hardened  
✅ Complete documentation  

**Built with:** Next.js, TypeScript, Tailwind CSS, Supabase, FastAPI
