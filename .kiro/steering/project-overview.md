# Talastock — Project Overview

## What is Talastock?
Talastock is a warm, modern inventory and sales dashboard built for Filipino SMEs,
especially businesses in Binondo and similar trading hubs. The name comes from
"Tala" (Tagalog for record/list) + "Stock".

## Tech Stack
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Python FastAPI
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth
- **Charts**: shadcn/ui Charts (built on Recharts)
- **PDF Export**: jsPDF
- **Deploy**: Vercel (frontend), Railway (backend)

## Target Users
- Filipino small-to-medium businesses
- Binondo trading and wholesale companies
- Sari-sari stores and retail shops
- Restaurant and food businesses

## Core Features
1. Product management (CRUD)
2. Inventory tracking with low stock alerts
3. Sales recording and history
4. Dashboard with charts and KPIs
5. PDF report export
6. Multi-category support
